# 积木城市建设社

这是一个保留原有经营玩法、但整体收敛为纯积木主题的静态城市建造游戏。

## 玩法

- 先铺道路，再放置住宅、商业、公共服务、校园、桥梁与地标。
- 建筑会随道路邻接、人口、人气、文化和交通状态产生变化。
- 角色会在街区中自动流动，带来轻微属性影响。
- 通过目标系统逐步扩张城市规模。

## 视觉

- 运行时素材位于 `assets/image2-clean/`。
- 当前版本使用本地积木化处理脚本强化塑料质感、描边和基座纹理。
- 不使用任何受保护品牌素材、名称或官方模型。

## 结构

- `app.js`：核心玩法与 UI 逻辑
- `styles.css`：界面与地图样式
- `data/`：建筑、角色、目标与事件数据
- `tools/lego_style_assets.py`：积木风素材处理脚本

## 验证

```bash
node --check app.js
node --check sfx.js
```
